# 🚀 AI Resume Analyzer BNP

An AI-powered Resume Analyzer that helps job seekers evaluate their resumes against a Job Description (JD). The application uses **Natural Language Processing (NLP)** techniques to calculate ATS scores, analyze skill gaps, compare resumes with job requirements, and provide personalized improvement suggestions.

## 🌐 Live Demo

🔗 https://ai-resume-analyzerbnp-9netteuxecrxkjzoyitphx.streamlit.app/

## 📌 Key Features

✅ ATS Score Calculation

✅ Resume & Job Description Matching

✅ Skill Gap Analysis

✅ Resume Section Analysis

✅ Personalized Improvement Suggestions

✅ Interactive Dashboard & Visualizations

✅ Downloadable Analysis Report

## 🛠️ Tech Stack

### 🐍 Python

Used for backend development, ATS logic, and data processing.

### 🎨 Streamlit

Used to create an interactive and responsive web application.

### 🤖 NLTK

Used for text preprocessing, tokenization, and NLP operations.

### 📚 Scikit-Learn

Used for TF-IDF Vectorization and Cosine Similarity calculations.

### 📈 Plotly

Used for ATS score charts, progress bars, and dashboard visualizations.

### 📄 PyPDF2

Used to extract text from uploaded resume PDFs.
